# Etiquetas de texto.


- Titulos (etiqueta h)
- Parrafos (Etiqueta p)
- Enlace (etiqueta a)
- Listas
    - ul

- Otras etiquetas de texto
    - Bold text (b)

    - Emphasized text (em)
    - Marked text (marl)
    - small  Small text (small)
    - Deleted t0ext (del)
    - Inserted text (ins)
    - Subscript text (sub)
    - Superscript text (sup)
